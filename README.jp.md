# STL/3MF フォトブース

![alt](preview.webp)

**[stl.guilliman.ru](https://stl.guilliman.ru)**

3Dプリント用のオンライン3Dモデルビューア。STLおよび3MFファイルをサポート。

## 機能

- 🎨 **マテリアル** — 19種類のプリセット（PLA、金属、ガラス、セラミックなど）
- 🖨️ **プリントベッド** — 50種類以上のプリンターモデル、サイズチェック
- 🌍 **ライティング** — 8種類の照明スキーム、8種類のHDRIマップ
- 📐 **寸法** — mm単位の正確なモデルサイズ
- 📷 **スクリーンショット** — シーンまたは透明背景付きのモデルをエクスポート
- 🌐 **言語** — EN, RU, ES, 中文, 日本語
- 💾 **永続化** — 設定がブラウザに保存される

## 操作方法

**マウス：**
- 左ボタン — 回転
- 右ボタン — パン（移動）
- スクロール — ズーム

**キーボード：**
- `1` — 等角図ビュー
- `2` — 正面ビュー
- `3` — 上面ビュー
- `4` — 側面ビュー

## 技術スタック

- Three.js
- WebGL 2.0
- ES6 Modules
